=============================================================================
DENODO TESTING TOOL: SAMPLE TEST SETS
=============================================================================

These are the Sample Test Sets for the Denodo Testing Tool.

These test sets are a quick introduction to the features and capabilities of
the Denodo Testing Tool. In order to execute them, you will need a working
installation of this software, as well as access to a Denodo Virtual DataPort
(VDP) installation.

Before executing these tests, you will need to read carefully and follow the
instructions at the data/README.txt file, and also have a look at the tool
configuration files in the samples1/conf and samples2/conf folders.

Once correctly configured, the two sample test sets should be executable from
the directory "$DENODO_TESTING_TOOL_INSTALLATION/samples" with:

$ $INST_DIR/bin/denodo-test.sh samples1/conf/configuration.properties samples1/tests/
$ $INST_DIR/bin/denodo-test.sh samples2/conf/configuration.properties samples2/tests/

Every test file contained in this test set includes comprehensive description
and comments. Also, the user manual for the Denodo Testing Tool is online in the 
following link:
https://community.denodo.com/docs/html/document/denodoconnects/9/en/Denodo%20Testing%20Tool%20-%20User%20Manual
